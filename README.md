 Humanness Tasks App

Android application for task recording with Kotlin and Jetpack Compose.

Features Implemented
1. Step 1: Start Screen with bilingual text (English + Hindi)
2. Step 2: Noise Test Screen with dB meter simulation
3. Step 3: Task Selection Screen with Material Design cards
4. Step 4: Text Reading Task with API integration
5. Step 5: Image Description Task 

 Technical Details
- Built with Kotlin and Jetpack Compose
- Material Design 3 components
- API integration with dummyjson.com
- Local task storage
- Press-and-hold recording functionality

 How to Run
1. Open in Android Studio
2. Build and run on emulator or device

 Project Structure
All code is in `MainActivity.kt` including:
- Start Screen
- Noise Test Screen  
- Task Selection Screen
- Text Reading Task
- Task History Screen
- Navigation setup
